# chornmeter_block_v3 项目复盘（机械秒表 + 掐表游戏）

## 项目定位
- 类型：高仿真机械秒表 + 反应速度游戏
- 目标：纯 CSS 绘制秒表，结合游戏化计时挑战
- 技术栈：纯 HTML/CSS/JS（单文件）
- 特点：拟物化设计 + 游戏评分系统

## 核心创新点

### 1) 拟物化秒表设计
- **金属表壳**：conic-gradient 实现金属光泽
- **玻璃反光**：135deg 线性渐变模拟玻璃
- **机械按钮**：:active 状态 + transform 模拟按压
- **连接件设计**：按钮与表壳之间的金属连接件

### 2) 双表盘系统
- **主表盘**：60秒一圈，红色秒针
- **子表盘**：30分钟一圈，黑色分针
- **刻度生成**：JS 动态生成，支持反向文字旋转

### 3) 掐表挑战游戏
- **随机目标时间**：根据难度生成 3-20 秒目标
- **评分系统**：基于误差计算分数（S/A/B/C/D 等级）
- **统计追踪**：最高分、挑战次数、平均误差
- **难度分级**：简单/普通/困难（影响时间范围和得分倍率）

### 4) 赛博朋克风格游戏面板
- **霓虹配色**：金色 (#ffd700) + 青色 (#00f5ff)
- **发光效果**：text-shadow + box-shadow 模拟霓虹灯
- **科技字体**：Orbitron 字体增强未来感
- **渐变按钮**：线性渐变 + 悬浮动效
