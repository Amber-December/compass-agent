# rational-course_v4 项目复盘（互动课件视角）

## 项目定位
- 类型：小学数学互动课件（转盘密码锁）
- 目标：通过故事化“寻宝”场景学习正负数与循环刻度
- 技术栈：Next.js + React + TS + Framer Motion

## 这个项目做得好的地方
1. **数学逻辑有“单一真相”**：`lockLogic.ts` 明确了方向语义、角度映射与取模。
2. **交互节奏合理**：练习页采用单步推进，降低学生认知负担。
3. **反馈机制完整**：正确/错误有明显视觉反馈，强化学习闭环。
4. **叙事化包装有效**：海盗宝藏主题增强参与感。
5. **结构可扩展**：页面与组件边界清晰，方便新增关卡。

## 关键经验（可迁移）
- 先锁定“数学规则”，再实现动画。
- 对环形状态一定做 normalize（处理负数与越界）。
- 交互页面尽量单屏，复盘页面可滚动。
- 单步演示必须保存“累计状态”（如累计旋转角），否则会重置。

## 可继续优化点
- 增加“讲解模式/挑战模式”切换（同内容不同引导强度）。
- 增加学习数据埋点（每步停留时长、错误类型、重播次数）。
- 增加无障碍支持（键盘操作、ARIA、色弱对比优化）。
- 抽离题目 DSL（步骤、答案、提示）以支持批量出题。

## 结论
这是一个“教学逻辑 > 视觉炫技”的正向样例。其核心资产不是单一页面，而是：
**环形数学模型 + 单步演示机制 + 反馈闭环**。
