# analytics

数据分析能力，负责从原始数据计算效能指标和趋势。

## 能力

- 计算项目核心 KPI（完成率、增长率、转化率等）
- 生成环比/同比趋势
- 识别异常波动和数据模式
- 输出结构化分析结果

## 分析维度

| 维度 | 指标 | 计算方式 |
|------|------|----------|
| 任务进度 | 完成率、延期率 | 已完成数 / 总数 |
| 达人增长 | 粉丝增长率、活跃度 | (本期 - 上期) / 上期 |
| 内容效果 | 播放量、爆款率、完播率 | 汇总/平均 |
| 商业变现 | GMV、ROI、转化率 | 汇总/比率 |
| 风险指标 | 合规事件、客诉率 | 计数/比率 |

## 输出格式

```json
{
  "project": "项目名",
  "period": "2026-W18",
  "kpis": {
    "gmv": { "value": 850000, "target": 800000, "status": "green" },
    "conversion_rate": { "value": 0.052, "target": 0.05, "status": "green" }
  },
  "trends": {
    "gmv": { "wow": -0.12, "mom": 0.08 }
  },
  "risks": []
}
```
