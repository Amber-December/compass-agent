# 直播运营培训
> MCN 事业部直播电商项目运营培训，覆盖选品、脚本、执行、复盘全流程。

---

## 一、直播前准备

### 1.1 选品会（每周三）

| 环节 | 负责人 | 输出物 |
| --- | --- | --- |
| 供应商对接 | 吕十五 | SKU 清单、库存确认、价格表 |
| 选品评审 | 郑十三 + 何十四 | 本周主推品、引流品、利润品 |
| 价格机制确认 | 吕十五 | 直播价、优惠机制、库存锁定 |

### 1.2 脚本准备

- **开场话术**（3min）：自我介绍、今日福利预告、关注引导
- **产品介绍**（5-8min/款）：痛点引入、产品展示、使用演示、价格揭晓
- **互动环节**：抽奖、问答、点赞目标、评论互动
- **收尾话术**：库存预警、最后福利、下播预告

### 1.3 设备检查清单

- [ ] 灯光亮度充足、无频闪
- [ ] 摄像头清晰、角度正确
- [ ] 推流设备网络稳定（备用 4G/5G）
- [ ] 商品链接已上架、价格正确
- [ ] 千川/DOU+投放计划已创建

## 二、直播中执行

### 2.1 角色分工

| 角色 | 人员 | 职责 |
| --- | --- | --- |
| 主播 | 郑十三 / 何十四 | 话术执行、产品讲解、粉丝互动 |
| 中控 | 吕十五 | 弹幕回复、商品上下架、节奏提醒 |
| 投流 | 施十六 | 千川/DOU+实时监控、预算调整 |

### 2.2 实时数据监控

| 指标 | 监控频率 | 异常处理 |
| --- | --- | --- |
| 在线人数 | 每 5min | 低于预期时，投流手追加预算 |
| 转化率 | 每 10min | 低于 3% 时，主播调整话术或换品 |
| ROI | 每 15min | 低于 1:2 时，暂停低效计划 |
| 弹幕情绪 | 持续 | 负面评论集中时，中控引导或主播回应 |

### 3.2 常见问题与对策

| 问题 | 原因 | 对策 |
| --- | --- | --- |
| 流量高但转化低 | 话术吸引力不足、价格敏感度 | 优化开场钩子、增加限时福利 |
| 转化率波动大 | 产品切换节奏不稳 | 固定每 15min 切换一款主推品 |
| ROI 持续下滑 | 投流人群包老化 | 每周更新人群包、测试新定向 |
| 退货率高 | 产品描述与实际不符 | 加强产品展示真实性、优化详情页 |
