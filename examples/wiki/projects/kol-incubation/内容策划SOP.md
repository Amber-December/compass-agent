## 二、选题流程
```plaintext
热点追踪 → 选题脑暴 → 初筛评审 → 达人匹配 → 脚本撰写 → 审核 → 拍摄排期

```

### 2.1 热点追踪（每周一）

- **抖音热榜**：前50条中与美妆/穿搭相关的热点
- **小红书趋势**：每周笔记增长最快的标签
- **竞品监测**：竞品MCN近7天爆款内容题材
- **AI辅助**：Notion AI生成热点关联选题建议

### 2.2 选题脑暴会（每周一下午）

参会：张三（主持）、各达人代表（线上）产出：本周15个候选选题

### 2.3 选题评审标准

| 维度 | 权重 | 说明 |
| --- | --- | --- |
| 热点关联度 | 30% | 是否与当前热点相关 |
| 差异化 | 25% | 与近期内容的重复度 |
| 达人匹配度 | 20% | 是否符合达人风格 |
| 商业潜力 | 15% | 是否有品牌合作空间 |
| 制作难度 | 10% | 是否可在48小时内完成 |

## 三、脚本撰写规范

### 3.1 开头3秒钩子（完播率关键）

| 类型 | 示例 | 适用 |
| --- | --- | --- |
| 反常识 | "用了10年洗面奶，你可能一直洗错了" | 教程 |
| 强对比 | "30元的粉底 vs 300元的粉底，结果..." | 测评 |
| 悬念 | "挑战用化妆品画一幅画，最后竟然..." | 变装 |
| 痛点 | "小个子女生夏天怎么穿显高？" | 穿搭 |

### 3.2 内容结构（15-90秒）
```plaintext
0-3秒   钩子（反常识/强对比/悬念/痛点）
3-8秒   问题提出或场景设定
8-20秒  核心内容展示
20-25秒 互动引导（"你遇到过吗？评论区告诉我"）
25-30秒 CTA（关注/点赞/收藏）

```

### 3.3 结尾引导

- 教程类："收藏起来，下次化妆用得上"
- 测评类："你用过这款产品吗？觉得怎么样？"
- 变装类："下一个变装挑战想看我cos谁？"
- 穿搭类："这套搭配适合什么场合？评论区聊聊"

## 四、审核流程

详见《内容审核规范》。

额外注意：

- 脚本需在拍摄前24小时提交审核
- 涉及品牌合作的内容需额外标注"合作"或"广告"
- 达人可在审核后提出1次修改意见

## 五、拍摄与剪辑

### 5.1 拍摄标准

| 项目 | 标准 |
| --- | --- |
| 分辨率 | ≥1080P |
| 帧率 | ≥30fps |
| 光线 | 均匀无阴影，肤色自然 |
| 背景 | 简洁，不抢主体 |
| 收音 | 清晰，无环境噪音 |

### 5.2 剪辑节奏

- 每3-5秒需有视觉变化（转场、字幕、特效）
- BGM选择：抖音热门音乐库，注意版权
- 字幕：关键信息加大加粗，品牌名标注

## 六、发布与数据回收

### 6.1 发布排期

| 平台 | 黄金时段 | 日更数量 |
| --- | --- | --- |
| 抖音 | 12:00-13:00, 18:00-20:00 | 2-3条 |
| 小红书 | 10:00-11:00, 20:00-22:00 | 1-2条 |
| B站 | 18:00-22:00 | 1条（长视频） |

### 6.2 数据监测（发布后24/48/72小时）

- 播放量、完播率、互动率（点赞/评论/转发/收藏）
- 粉丝净增、粉丝画像变化
- 评论区关键词分析（正面/负面/建议）

### 6.3 爆款复盘

播放量超过达人平均3倍的内容，需在48小时内完成复盘文档：

- 爆款原因分析
- 可复制要素提取
- 同类型选题储备
