# 内容审核培训
> MCN 事业部内容安全与合规培训，适用于所有参与内容创作、发布、审核的同学。

---

## 一、审核流程
```plaintext
创作者自审 → 项目 PM 审核 → 合规审核（如需） → 发布

```


| 环节 | 负责人 | 重点 | 耗时 |
| --- | --- | --- | --- |
| 自审 | 创作者 | 合规、版权、品牌形象 | 10min |
| PM 审核 | 项目 PM | 内容方向、数据预期、品牌匹配度 | 20min |
| 合规审核 | 法务/合规团队 | 广告法、平台规则、敏感内容 | 1-2h |
| 发布 | 运营/创作者 | 定时发布、数据监测 | 5min |

---

## 二、审核清单

### 2.1 通用检查项（所有内容）

- [ ] 无政治、宗教、色情、暴力内容
- [ ] 无未经审核的医疗、金融、保健品广告
- [ ] 无竞品对比中贬低他人的内容
- [ ] 无侵犯知识产权的音乐、视频素材
- [ ] 无虚构数据、伪造用户评价
- [ ] 广告法极限词检查（"最好""第一""绝对"等）

### 2.2 短视频专项检查

- [ ] 画面无违规品牌 logo 或水印
- [ ] BGM 有版权或使用平台曲库
- [ ] 口播内容与字幕一致
- [ ] 商品链接与视频内容匹配

### 2.3 直播专项检查

- [ ] 话术脚本已预审
- [ ] 价格、库存、优惠机制与商品页一致
- [ ] 直播间背景无违规元素
- [ ] 实时弹幕监控机制已启动

---

## 三、禁区清单

| 禁区 | 说明 | 处罚 |
| --- | --- | --- |
| 政治敏感 | 涉及政治人物、敏感事件、民族宗教 | 立即下架，视情节处理 |
| 虚假宣传 | 夸大功效、伪造数据、虚假承诺 | 立即下架，品牌方追责 |
| 侵权内容 | 盗用音乐、视频、图片、文案 | 立即下架，赔偿损失 |
| 低俗内容 | 性暗示、暴力、恶心、恐怖 | 立即下架，账号限流 |
| 竞品攻击 | 公开贬低、对比抹黑其他品牌 | 立即下架，商务追责 |

## 四、案例分析

### 案例 1：极限词违规

**情况**：某达人视频中口播"这款面膜是全网最好的"。

**处理**：视频下架，重新剪辑后发布，达人罚款 500 元。

**教训**：口播脚本需提前审核，避免使用绝对化用语。

### 案例 2：音乐侵权

**情况**：短视频使用未经授权的流行音乐，被平台下架。

**处理**：更换为平台曲库音乐，重新发布，流量损失 50%。

**教训**：统一使用平台曲库或购买商用授权音乐。
