# 品牌合作SOP
> MCN 事业部品牌合作全流程操作手册，覆盖需求对接、达人匹配、方案策划、报价谈判、合同签署到复购跟进。

---

## 一、合作流程
```plaintext
需求对接 → 达人匹配 → 方案策划 → 报价谈判 → 合同签署 → 内容制作 → 发布推广 → 数据回收 → 结算付款 → 复购跟进

```

---

## 二、各阶段操作规范

### 2.1 需求对接（1-3天）

**输入**：品牌方Brief**输出**：《品牌需求确认单》

确认内容：

- [ ] 品牌背景与产品信息
- [ ] 合作目标（曝光/转化/种草）
- [ ] 目标受众画像
- [ ] 内容形式要求（短视频/图文/直播）
- [ ] 数量与质量要求
- [ ] 时间节点
- [ ] 预算范围
- [ ] 竞品排除清单

### 2.2 达人匹配（1-2天）

**依据**：《达人商业潜力评估表》

匹配维度：

| 维度 | 权重 | 说明 |
| --- | --- | --- |
| 粉丝画像匹配度 | 30% | 品牌目标受众 vs 达人粉丝画像 |
| 内容调性契合度 | 25% | 品牌调性 vs 达人内容风格 |
| 历史合作表现 | 20% | 过往商单数据、品牌方评价 |
| 配合度 | 15% | 达人沟通响应速度、修改配合度 |
| 报价合理性 | 10% | 性价比评估 |

**达人报价参考**：

| 达人等级 | 粉丝量 | 短视频报价 | 图文报价 | 直播专场报价 |
| --- | --- | --- | --- | --- |
| S级 | ≥10万 | 3-5万 | 1-2万 | 8-15万 |
| A级 | 3-10万 | 1-3万 | 0.5-1万 | 3-8万 |
| B级 | 1-3万 | 0.5-1万 | 0.2-0.5万 | 1-3万 |
| C级 | <1万 | 0.1-0.5万 | 0.05-0.2万 | 暂不开放 |

### 2.3 方案策划（2-3天）

**输出**：《品牌合作方案PPT》

方案内容：

1. 项目背景与目标
1. 达人矩阵与分工
1. 内容规划（选题+脚本方向）
1. 发布排期
1. 预期效果（曝光量/互动量/转化率预估）
1. 报价明细
1. 风险评估与预案

### 2.4 报价谈判（3-7天）

| 场景 | 策略 | 底线 |
| --- | --- | --- |
| 首单合作 | 让利10%，换取长期框架协议 | 不低于成本价 |
| 复购合作 | 按正常报价，可赠送增值服务 | 不低于首单价格 |
| 大单合作（≥10万） | 分期付款，降低品牌方风险 | 首付≥30% |
| 紧急合作（<7天） | 加急费20% | 不影响其他项目 |

### 2.5 合同签署（2-5天）

**合同要素**：

- 合作内容与数量
- 交付标准与验收流程
- 付款方式与时间节点
- 知识产权归属
- 保密条款
- 违约责任

**付款节点**：

| 节点 | 比例 | 条件 |
| --- | --- | --- |
| 签约 | 30% | 合同签署后3个工作日内 |
| 内容交付 | 50% | 所有内容通过品牌方验收 |
| 结算 | 20% | 发布完成后7个工作日内 |

### 2.6 内容制作与发布

制作流程：
```plaintext
脚本撰写 → PM审核 → 品牌方确认 → 拍摄制作 → 内部审核 → 品牌方终审 → 修改定稿 → 发布

```

发布前检查：

- [ ] 内容已标注"合作"或"广告"
- [ ] 品牌方已书面确认可发布
- [ ] 发布时间已同步品牌方

### 2.7 数据回收与结算

**数据指标**：

| 指标 | 达标线 |
| --- | --- |
| 曝光量 | ≥合同约定的80% |
| 互动率 | ≥3% |
| 完播率 | ≥40% |

**达人分成**：

- 独家签约达人：60%
- 非独家签约达人：50%
- 项目制合作：按单协商

### 2.8 复购跟进

- 结算后3天内：发送感谢信+数据总结
- 结算后1周：询问合作满意度+收集反馈
- 结算后1月：推送新达人/新合作方案
- 每季度：发送项目数据报告，维持关系

## 三、常见问题

**Q：品牌方要求修改内容，达人不愿意怎么办？**A：首先判断修改是否合理。合理修改达人必须配合。不合理修改（如贬低竞品）由PM出面与品牌方沟通，提供替代方案。

**Q：品牌方拖延付款怎么办？**A：合同明确付款节点和违约金。拖延超过15天，暂停该品牌所有新合作。拖延超过30天，法务介入。

**Q：达人私自接商单怎么处理？**A：首次警告+扣除当月绩效。第二次扣除季度奖金+降级。第三次解约+追究违约责任。
