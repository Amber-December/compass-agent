# 数据表结构定义

本目录存放飞书 Base 数据表的 schema 定义，用于代码生成和数据校验。

## 表清单

| 表名 | 说明 | 状态 |
|------|------|------|
| project_kpi | 项目核心 KPI 数据 | 待创建 |
| task_list | 任务列表与完成状态 | 待创建 |

| weekly_report | 周报元数据 | 待创建 |
| risk_log | 风险记录与处理 | 待创建 |

## 命名规范

- 表名使用 snake_case
- 字段名使用 snake_case
- 时间字段统一使用 ISO 8601 格式（如 2026-05-01T00:00:00Z）
- 金额字段统一使用整数（分）或保留两位小数
