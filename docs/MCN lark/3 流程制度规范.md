# MCN 流程制度规范

## 1. 达人管理规范

### 1.1 达人签约流程
1. 素人筛选（粉丝量/内容质量/人设匹配度）
2. 试合作期（1-2 周，测试内容产出能力）
3. 正式签约（合同签署、账号绑定、分成比例确认）
4. 孵化扶持（内容培训、流量扶持、商单对接）

### 1.2 达人考核标准
- **月度考核**：内容产出数量、粉丝增长、商单完成率
- **季度评估**：头部达人晋升、尾部达人淘汰或转岗
- **违规处理**：内容违规、商单违约、私自接单的处罚机制

## 2. 内容审核规范

### 2.1 审核流程
1. **自审**：创作者自查（合规、版权、品牌形象）
2. **项目审核**：PM 审核（内容方向、数据预期）
3. **合规审核**：法务/合规团队审核（广告法、平台规则）
4. **发布**：定时发布、数据监测

### 2.2 禁区清单
- 涉及政治、宗教、色情、暴力的内容
- 未经审核的医疗、金融、保健品广告
- 竞品对比中贬低他人的内容
- 侵犯知识产权的音乐、视频素材

## 3. 商务合作规范

### 3.1 品牌合作流程
1. 线索收集（商务拓展、品牌主动询盘）
2. 需求沟通（品牌目标、预算、时间节点）
3. 方案策划（达人匹配、内容创意、报价）
4. 合同签署（法务审核、付款条款）
5. 执行与交付（内容制作、数据回收、结案报告）

### 3.2 报价与分成
- **达人报价**：根据粉丝量、互动率、历史 ROI 定价
- **平台分成**：星澜传媒 vs 达人的分成比例（按合同）
- **返点政策**：长期合作品牌的返点机制

## 4. 私域运营规范

### 4.1 私域沉淀流程
1. **引流**：短视频/直播间引导加入粉丝群
2. **承接**：社群欢迎语、人设定位、价值承诺
3. **激活**：定期活动、专属福利、互动话题
4. **转化**：商品链接、直播预告、限时优惠
5. **留存**：会员体系、积分商城、专属内容

### 4.2 社群管理规范
- 群规制定与公示
- 广告/刷屏管理机制
- 用户投诉处理流程
